# mtgcalc

A clean mortgage calculator for comparing conventional, FHA, VA, and USDA loan scenarios.

## What it does

- compares four common loan types
- supports down payment by dollar amount or percent
- estimates principal and interest
- estimates monthly mortgage insurance where applicable
- gives a fast side-by-side style scenario tool for rough planning

## Stack

- Vue 2
- BootstrapVue
- Vue CLI

## Local development

```bash
npm install
npm run serve
```

## Production build

```bash
npm run build
```

## Notes

This is a quick-estimate tool, not a full underwriting calculator.
Taxes, homeowners insurance, HOA dues, and loan-specific fees are not included.
